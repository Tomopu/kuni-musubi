# Kuni-Musubi Research & Documentation

本ディレクトリは、「Kuni-Musubi（クニムスビ）」の立ち上げに向けた、技術・市場・学術的背景の調査資料をまとめたものです。

## 📁 ドキュメント構成

### 1. [product/](./product/)
プロダクトの核となるビジョン、ターゲット層、提供価値など、Kuni-Musubiの「根幹」と「Why」を定義するドキュメントをまとめたディレクトリです。

### 2. [research/](./research/)
技術仕様の実現性、市場環境、学術的な裏付けなど、開発の土台となる多角的なリサーチ結果をストックするディレクトリです。

### 3. [technical/](./technical/)
フロントエンドおよびバックエンドの実装方針、アーキテクチャ設計、データベース選定など、具体的なエンジニアリングに関する仕様をまとめるディレクトリです。
