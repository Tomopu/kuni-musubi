# ペルソナ設定

作成者: 泉 知成  
作成日: 2026年4月29日  
更新日: 2026年4月29日

## 結論

Kuni-Musubi の主なペルソナは、**「政治への苦手意識もち、政府への不信感を持つ、20代中盤の若手社会人」** とする。

学生も重要な対象ではあるが、初期プロダクトで最も検証すべきユーザーは、税金・物価・賃金・社会保障・将来不安をすでに自分の生活課題として感じている若手社会人であると考えた。この層は政治を「遠い話」と感じつつも、生活への影響は日々実感しているため、ポジティブで建設的な政治ニュースによって認知変容が起きやすい。

本プロダクトの目指すところは、ポジティブで建設的な政治ニュースによって、政府や公共への信頼感を回復させ、日本に対する希望・愛国心を持てるようにすることである。ここで、Kuni-Musubi では「愛国心」を政府への無条件の信頼や国家主義ではなく、**日本の社会・文化・制度に誇りをもち、より良くしたいと思える国民的な精神状態** として定義する。
(プロパガンダにならないように細心の注意を払う。)

## 主ペルソナ

### 基本プロフィール

| 項目 | 内容 |
| :--- | :--- |
| 年齢 | 25歳 |
| 性別 | 男性 / 女性 |
| 居住地 | 首都圏の賃貸マンション |
| 職業 | IT・人材・広告・小売本部などの若手会社員 |
| 年収 | 320万〜430万円 |
| 家族構成 | 独身、一人暮らし。実家は地方 |
| 政治知識 | 選挙には行ったことがあるが、政党や政策の違いは説明できない |
| 政治姿勢 | 支持政党なし。選挙で投票した政党がちゃんと役割を果たしているか疑問に思っている。政府・政治家・既存メディアに漠然とした不信感がある。 |
| 情報接触 | X、Instagram、TikTok、YouTube、Yahoo!ニュース、LINE NEWS |

### 一言で表すと

「政治はよくわからない。でも、給料が増えないこと、税金や社会保険料が重いこと、将来が不安なことには腹落ちしている。SNS で流れてくる政治批判を見るたびに、日本はもう良くならないのではないか、日本は『オワコン』なのではないかと思ってしまう若手社会人」

### 生活背景

平日は仕事で疲れており、政治ニュースをじっくり読む余裕はない。ニュースを見る時間は、通勤中、昼休み、就寝前のスマホ時間が中心。社会人になってから、給与明細、家賃、物価高、奨学金返済、将来の結婚・出産・介護への不安を通じて、政治が生活に関係していることは感じ始めている。

一方で、政治ニュースは難しく、対立的で、怒っている人ばかりに見える。SNS では「政府は何もしていない」「日本は終わっている」「日本よりも海外を優遇している」「若者は搾取されている」といった投稿に多く触れており、それを完全に信じているわけではないが、反論できるだけの知識もない。

### 心理状態

| 項目 | 内容 |
| :--- | :--- |
| 表層ニーズ | 政治ニュースを短く、わかりやすく、嫌な気持ちにならずに知りたい |
| 深層ニーズ | 自分の生活が少しでも良くなる可能性を感じたい |
| 不安 | 何を信じればいいかわからない。自分だけ知らないまま損をしている気がする |
| 不信 | 政治家は自分たちのことを見ていない。メディアは切り取りが多い。SNSも煽りが多い |
| 欲しい感情 | 「全部ダメ」ではなく、「改善しようとしている人や政策もある」と思えること |
| 避けたい体験 | 説教される、思想を押しつけられる、専門用語で置いていかれる、政党広報のように見える |

### 現在の行動

1. X や TikTok で政治家の失言、炎上、増税批判、社会保障批判を見る。
2. 気になる話題があっても、検索すると難しい記事や対立的な意見ばかり出てきて読むのをやめる。
3. 選挙前だけ候補者や政党を調べようとするが、比較軸がわからず、結局「なんとなく」で判断する。
4. 日本の文化、治安、食、旅行先としての魅力には愛着があるが、「政府を信じる」「国に誇りを持つ」と言われると抵抗がある。

### Kuni-Musubi を使う理由

| 期待 | プロダクトで満たすべき内容 |
| :--- | :--- |
| 政治ニュースで疲れたくない | 怒りや対立を煽る見出しではなく、政策の進展や改善例を中心に見せる |
| 難しい言葉を避けたい | 1記事ごとに「何が変わるのか」「自分の生活への影響」を平易に説明する |
| 偏った情報に飲まれたくない | 一次情報、複数メディアの論点、残る課題をセットで表示する |
| 自分の考えを持ちたい | 閲覧傾向から関心テーマを可視化し、政策比較への導線を作る |
| 日本に希望を持ちたい | 政党を問わず、実際に社会を良くしようとする取り組みを見せる |

## 副次ペルソナ

### 大学生・専門学生

| 項目 | 内容 |
| :--- | :--- |
| 年齢 | 18〜22歳 |
| 主な関心 | 学費、奨学金、就職、最低賃金、ジェンダー、教育、気候変動、住民票と投票手続き |
| 政治との距離 | 授業やSNSで断片的に触れるが、自分ごと化しづらい |
| 障壁 | 用語が難しい、政党の違いがわからない、住民票や不在者投票が面倒、政治は自分には関係ない、選挙は面倒だから行かない。 |
| 有効な体験 | 「はじめての選挙」「政党の超基本」「争点の3分解説」「投票手続きナビ」 |

学生向けには、政治ニュースそのものよりも、政治リテラシーの入口を厚くする必要がある。政党名、国会の仕組み、選挙制度、住民票、不在者投票、政策比較などの基礎コンテンツが重要になると考える。

### 20代後半〜30代前半の生活実感層

| 項目 | 内容 |
| :--- | :--- |
| 年齢 | 27〜34歳 |
| 主な関心 | 手取り、住宅、結婚、子育て、保育、医療、介護、年金、転職 |
| 政治との距離 | 政治に不満はあるが、ニュースを見ると疲れる |
| 障壁 | 忙しい、ニュースが暗い、自分の生活に関係する政策を探しづらい |
| 有効な体験 | 「生活への影響」タグ、政策のメリット・課題、自治体や地域政策への導線 |

この層は主ペルソナよりもさらに生活課題が明確である。初期ターゲットを広げる場合は、手取り、子育て、住宅、社会保障を中心にした編集が有効になる。

## 学生か社会人かによる情報設計の違い

| 観点 | 学生向け | 若手社会人向け |
| :--- | :--- | :--- |
| 主な問い | 政治って何から知ればいいのか | この政策は自分の生活にどう関係するのか |
| 重要テーマ | 学費、奨学金、就活、最低賃金、投票方法、気候変動 | 手取り、税金、社会保険料、物価、賃上げ、住宅、将来不安 |
| 必要なコンテンツ | 用語解説、政党の基本、選挙制度、投票ナビ | 生活影響の要約、政策の進捗、財源・課題、複数政党の取り組み |
| UIの入口 | 学ぶ、診断する、選挙前に確認する | 毎日軽く読む、生活テーマで追う、希望のある動きを知る |
| 継続理由 | 自分の考えを作れる | 生活不安が少し整理される |
| 離脱理由 | 難しい、説教っぽい、学校の教材のよう | 忙しい、長い、結局プロパガンダに見える |

初期プロダクトでは、若手社会人向けに「生活への影響」と「建設的な進展」を中心に置く。学生向けの基礎コンテンツはオンボーディングや選挙前特集として段階的に追加する。

## 掲載すべき情報

### 必須

1. 各政党・行政によるポジティブな取り組み
2. その政策が解決しようとしている社会課題
3. ユーザーの生活への具体的な影響
4. 財源、実現可能性、副作用などの残る課題
5. 与野党・複数メディア・専門家による建設的な論点
6. 政党の理念・主要政策の平易なまとめ

### 主ペルソナ向けに優先するテーマ

1. 賃上げ、雇用、働き方
2. 税金、社会保険料、手取り
3. 物価高、エネルギー、家計支援
4. 住宅、地方移住、都市生活コスト
5. 少子化、結婚、子育て支援
6. 医療、年金、介護の将来不安
7. 治安、防災、外交・安全保障の生活影響

### 表現上の注意

「政府は正しい」「日本を愛そう」といったメッセージは避ける。主ペルソナは、上から価値観を与えられることに敏感である。アプリの態度は、愛国心を直接説くのではなく、各政党や行政、市民、企業が日本を良くしようとしている具体例を積み上げ、ユーザーが自分で「まだ希望はある」と感じられる設計にする。

## アンチペルソナ

初期ターゲットから外すべきユーザーは以下。

1. すでに政治的立場が強く固まっており、反対意見を見る目的が論破や攻撃であるユーザー
2. 政治ニュースを長文で深く読み込みたい専門家・政治マニア
3. 特定政党の宣伝だけを求めるユーザー
4. 陰謀論や極端な怒りの共有を目的にニュースを見るユーザー

Kuni-Musubi は、強い思想対立の場ではなく、政治から距離を置いている人が心理的安全性を保ちながら再接近するための入口である。

## プロダクト要件

1. オンボーディングでは、支持政党を必須にしない。「支持政党なし」「よくわからない」を自然な選択肢として扱う。
2. 記事カードには「生活への影響」「進展したこと」「残る課題」を短く表示する。
3. 反対側の政党ニュースを混ぜる場合は、「異なる視点の建設的な取り組み」のように明示する。
4. 多様性の比率はユーザーが調整できるようにする。
5. 政治用語はタップで説明し、本文を読まなくても要点がわかる設計にする。
6. ダッシュボードは専門的な分析画面ではなく、「自分は何に関心があるのか」が直感的にわかる自己理解ツールにする。
7. ポジティブニュースだけで完結させず、必ず課題や論点を併記し、プロパガンダに見えるリスクを下げる。

## ペルソナステートメント

Kuni-Musubi は、政治に詳しくないまま SNS の批判的な情報に触れ続け、日本の未来に希望を持ちにくくなっている20代の若手社会人に向けて、各政党や行政の建設的な取り組みをわかりやすく届ける。ユーザーが政治を「怒りと対立の世界」ではなく、「自分の生活と日本の未来を少しずつ良くするための手段」として捉え直せる状態を目指す。
