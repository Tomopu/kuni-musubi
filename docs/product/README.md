# Product Documents

Kuni-Musubi のプロダクト方針、ターゲット、提供価値などをまとめたドキュメントです。

## ドキュメント一覧

- [20260428_プロダクト概要.md](20260428_プロダクト概要.md)
  - Kuni-Musubi の名称、ビジョン・ミッション、開発背景、解決したい課題、提供価値、コア機能、UI/UX の方向性をまとめたドキュメント。

- [20260429_ペルソナ設定.md](20260429_ペルソナ設定.md)
  - 初期プロダクトで想定する主ペルソナ・副次ペルソナ、学生/社会人による情報設計の違い、掲載すべき情報、プロダクト要件を整理したドキュメント。

- [20260430_画面設計.md](20260430_画面設計.md)

- [20260430_記事カード表示設計.md](20260430_記事カード表示設計.md)
  - ホーム画面に表示する記事カードの表示項目、タイトル・要約・タグ・サムネイル・日付の扱いを整理したドキュメント。

- [20260430_記事詳細画面表示設計.md](20260430_記事詳細画面表示設計.md)

- [ui-designs/](ui-designs/)
  - GPT images などで作成した PNG の UI デザイン参照、マスコット素材、各画像の注釈を管理するディレクトリ。UI 実装・レビュー時は、既存の画面設計ドキュメントとあわせて参照する。
  - 画像だけでは意図を読み違えやすいため、`images/` に PNG を置き、対応する注釈 Markdown を `notes/` に置く。1枚に複数画面が含まれる場合は、左から順に対象画面を明記する。
